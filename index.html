Portfolio Verwalter

Ich habe ein kleines Finanztool mit Python gebaut welches über eine Bibliothek aktuelle Finanzkurse abfragen kann, diese in Diagrammen sowie Zahlen darstellen kann. Man kann sein eigenes Portfolio erstellen und verwalten. Das Programm ermöglicht es ebenfalls die selben Titeln zu verschiedenen Preisen zu kaufen. Die Titel im Portfolio werden lokal in einer Datenbank gespeichert.

Tech: Python, Matplotlib, Pandas, Tkinter, yFinance
